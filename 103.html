Project Description

I wanted to create an interactive Chart to see how many players where injured per team and matchday
My I pulled all the data from Transfermarkt.de using the Node library puppeteer.

First I got all the teams from

https://www.transfermarkt.de/1-bundesliga/startseite/wettbewerb/L1/plus/?saison_id=2018

with the following short script: get_club_list.js


Than I tried to get all players of each team estimating that none had more than 47 (because that was the biggest roster) on there roster with this script(link to GitHub)

(This script was also a bit more difficult because it had to write data to an output file and use data from input file using the node module fs

I have done that a view times before. But its always another source of errors

The output produced a file with about 600 entries


When i got all the players profiles
I first used this script to go to each players injury history and draw every entry from the 18/19 season with start and end date and name of injury along side with the team.
Two problems occured with that in the output file.
A lot of players changed the club during the season and my lack of excel skills.

The challenge to detect if match days were within a time of injury was not solveable for me at this time.
(If you have an idea how to do that let me know)

So I wrote a a 4th script(linkt to Github).

This one went to the performance page of each player
and got every missed game caused by injury of each player counting them up in an excel file later.

A crucial question was if i had to leave players aside manually because they wouldn't have played even if they weren't injured. But transfermarkt.de just put the Bundesliga on the performance page if they have been in the squad at least once.
I just took data of players who have been in the squad more than once.

So afterwards I had an output file which was much easer to analyze.
So I could make a table with the number of injured players for each club on each match day.

So I could start to make a chart.
I just had gotten to know highcharts.
Most difficult for me was to inject the data from a csv. I didn't want to make manually entries because I was planning on doing he analysis for the other 3 big leagues in Europe as well.

But in the end i did make a chart with manual data injection so that i could use some images to make it look better
Also I pulled the data of the Premier League and made a simple chart for them as well
